Sandrine PHAM — Portfolio Développeuse Web Freelance

Portfolio professionnel développé avec React pour présenter mes projets, mes compétences et ma manière de concevoir des sites web sur-mesure.

Le site a été pensé comme une démonstration concrète de mon approche du développement :
structure claire, interfaces modernes, expérience utilisateur fluide et adaptation visuelle selon le contexte du projet.

Trois univers graphiques permettent d’explorer différentes directions de design tout en conservant une base technique commune.

Aperçu du projet

Ce portfolio présente :
mes projets professionnels, personnels et de formation
mes compétences techniques front-end et full-stack JavaScript
ma méthode de travail et mon approche de conception web
un système de thèmes visuels interactifs

L’objectif est double :
présenter mon travail à des clients potentiels
permettre à des développeurs d’explorer mon code et mon approche technique

Installation du projet

Cloner le repository :
git clone https://github.com/SandrinePham/portfolio.git

Installer les dépendances :
npm install

Lancer le projet en développement :
npm run dev

Le site sera accessible sur :
http://localhost:5173
Déploiement

Le projet peut être déployé sur :
Vercel
Netlify
GitHub Pages

La configuration actuelle utilise HashRouter, ce qui facilite le déploiement sur GitHub Pages.
